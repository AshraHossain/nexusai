Contract Review Workflow

Reuses KnowledgeOps + SentinelAI + EvalOps — zero duplicate code

📚
Retrieve Precedents
KnowledgeOps pulls relevant contracts, legal standards, and precedents from the contracts collection
⚖️
Contract Analysis
Extracts parties, obligations, critical clauses, non-standard terms, timelines, and financial terms
🔬
Legal Research
Compares clauses against retrieved standards. Identifies regulatory red flags and compliance deviations
🎯
Risk Scoring
SentinelAI + ContractRiskAgent produce 0–100 risk score. Score > 40 triggers human approval gate
👤
Human Approval
LangGraph interrupt_before pauses workflow. POST /approvals/{id}/decide to approve or reject
📋
Executive Summary
ContractSummaryAgent produces polished report: risk level, negotiation points, approval recommendation

Usage

POST /workflows/run
{
  "request": "Review this SaaS agreement for IP ownership and limitation of liability clauses...",
  "workflow_type": "contract_review",
  "knowledge_collection": "contracts"
}

AI SDET Workflow

Perfect for your background — generate tests, Playwright, API tests, and defect analysis from a feature description

📖
Retrieve Specs
KnowledgeOps retrieves existing tests, specs, and documentation relevant to the feature
📋
Requirements Analysis
Extracts functional/non-functional requirements, acceptance criteria, edge cases, and test strategy

pytest Unit Tests
Full pytest suite with fixtures, parametrize, async support, AAA pattern, coverage for all paths

Playwright E2E
Page Object Model, desktop + mobile viewports, screenshot on failure, expect() assertions
🌐
API Tests
httpx + pytest async API tests covering all HTTP methods, status codes, schema validation, auth
🐛
Defect Analysis

  "workflow_type": "sdet",
  "knowledge_collection": "default"
